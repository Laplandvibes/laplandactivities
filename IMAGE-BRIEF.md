## Production notes

- **Format:** Save as **WebP** at 80 % quality. Fallback PNG only if your tool refuses WebP — we'll convert.
- **Aspect ratios:** Heroes 16:9 (1920×1080). Cards 4:3 or 3:2 at 1200×900. OG strictly 1200×630.
- **No text under any circumstances.** If your generator slips a logo in, regenerate. We add titles in HTML.
- **Filename = exact match** to the names above. The data layer auto-loads them — no code change needed when you drop a file in.
- **Size budget:** target ≤ 250 kB per hero, ≤ 120 kB per card. If files come out heavier, send raw and I'll re-encode.
- **Color rule:** keep imagery in the deep-night family (cool blues, snow whites, aurora green, occasional warm window light). Avoid bright commercial blue skies — they fight the LV brand.
